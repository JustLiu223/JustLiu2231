# 单例模式创建

| 步骤 | 操作                      | 目的                 |
| ---- | ------------------------- | -------------------- |
| 1️⃣    | `X_ASSERT(pMainApp);`     | 检查指针是否有效     |
| 2️⃣    | `IsKindOf(...)`           | 检查对象类型是否匹配 |
| 3️⃣    | `Start()`                 | 启动线程或任务       |
| 4️⃣    | `Join(NULL)`              | 等待任务执行完成     |
| 5️⃣    | `DestroyRunnableObj(...)` | 安全释放对象内存     |
| 6️⃣    | `pMainApp = NULL;`        | 避免野指针           |
